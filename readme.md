# Portafolio Personal

## Descripción
Este proyecto corresponde a un portafolio web desarrollado como parte de la evaluación de Desarrollo Front-End. Presenta información personal, habilidades, proyectos y un formulario de contacto.

## Características
- Navegación mediante navbar con enlaces internos
- Sección "Sobre mí"
- Listas y tabla de habilidades
- Galería de proyectos con imágenes
- Formulario de contacto
- Diseño responsive adaptable a distintos dispositivos

## Tecnologías utilizadas
- HTML5
- CSS3
- Git
- GitHub

## Estructura del proyecto
- index.html
- style.css
- README.md
